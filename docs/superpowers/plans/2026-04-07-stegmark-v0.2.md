# StegMark v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the self-hosted `hidden` engine path to StegMark with PyTorch model definitions, training/export scaffolding, optional runtime integration, and tests.

**Architecture:** Keep `v0.1` usable as the default path while layering in the `HiDDeN`-style workflow behind optional training/runtime dependencies. The package should gain well-bounded `nn`, `training`, `data`, and `hidden` runtime modules without making the base CLI unusable when model artifacts are absent.

**Tech Stack:** Python 3.10+, NumPy, Pillow, Click, Rich, optional PyTorch/TorchVision, optional ONNX Runtime

from stegmark.nn.hidden.discriminator import HiddenDiscriminator
from stegmark.nn.hidden.encoder import HiddenEncoder


def test_hidden_encoder_output_shape() -> None:
    encoder = HiddenEncoder(message_bits=32)
    image = torch.rand(2, 3, 128, 128)
    message = torch.randint(0, 2, (2, 32), dtype=torch.float32)
    encoded = encoder(image, message)
    assert encoded.shape == image.shape


def test_hidden_decoder_output_shape() -> None:
    decoder = HiddenDecoder(message_bits=32)
    image = torch.rand(2, 3, 128, 128)
    logits = decoder(image)
    assert logits.shape == (2, 32)


def test_hidden_discriminator_output_shape() -> None:
    discriminator = HiddenDiscriminator()
    image = torch.rand(2, 3, 128, 128)
    scores = discriminator(image)
    assert scores.shape == (2, 1)
