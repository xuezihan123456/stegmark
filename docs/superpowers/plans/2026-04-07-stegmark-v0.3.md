# StegMark v0.3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a usable benchmark suite with image attacks, structured benchmark results, and a `stegmark benchmark` CLI command.

**Architecture:** Keep attacks isolated under `evaluation/attacks.py` and benchmark orchestration under `evaluation/benchmark.py`. The benchmark path should use the existing public service/engine abstractions, default cleanly to `native`, and degrade explicitly when requested engines are unavailable.

**Tech Stack:** Python 3.10+, NumPy, Pillow, Click, Rich
